# ntt-loggable-aop

Java Method Logging with AOP and Annotations. - AOP with Spring Boot

Lo scopo è quello di tracciare\loggare i tempi di esecuzioni dei java method. 

### Dettagli implementazione 

* Annotation with example metadata: [Loggable.java](https://github.com/VincenzoNTT/ntt-loggable-aop/blob/master/src/main/java/com/nttdata/aop/Loggable.java)
* Implementation of different advice types: [LogInterceptor.java](https://github.com/VincenzoNTT/ntt-loggable-aop/blob/master/src/main/java/com/nttdata/aop/LogInterceptor.java)
